# /auto-commit

당신은 시니어 개발자입니다. 현재 스테이징된 파일들을 각각 분리해서 커밋해야 합니다.

## 지시사항
1. 터미널 도구를 사용해 `git diff --cached --name-only`를 실행하여 스테이징된 파일 목록을 가져오세요.
2. 각 파일에 대해 다음 과정을 반복하세요:
   a. `git diff --cached <파일명>`으로 해당 파일의 변경점만 읽어옵니다. (전체 파일 읽기 금지)
   b. 변경 내용을 분석하여 Conventional Commits 형식(예: `[FEAT] ...`, `[FIX] ...`)으로 1줄짜리 커밋 메시지를 생성합니다.
   c. 터미널 도구를 사용해 해당 파일만 따로 커밋합니다: `git commit -m "생성한 메시지" <파일명>`
3. 모든 파일의 커밋 처리가 완료되면, 어떤 파일들이 어떤 메시지로 커밋되었는지 깔끔한 표 형태로 사용자에게 보고하세요.
